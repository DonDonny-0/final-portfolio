# PROJECT GOAL
My Software Engineering Portfolio which displays a list of my software engineering projects, including live demos and public GitHub Repositories.
## Core Features
- Admin Login
- Create Project
- Edit Project
- Delete Project
- Add Skill
- Edit Skill
- Delete Skill
- Post message through SMTP
## Tech Stack and Structure
### Frontend
- React
- Router
- State Management (Redux)
### Backend
- Laravel API
- Laravel Sanctum
- MySQL
## Database Migrations
- Users (admin)
- Projects
- Skills
